# Hotel Project
# How to use
## Start back-end server
First, direct to backend folder  
 ``` 
  cd backend 
```
### First time running 
When you running server for the first time, you need to install package by using :  
```
  npm install 
```
Make sure that you have installed npm.
### Starting backend server  
```
  npm start
```
